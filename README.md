# CairnBooks

> Cairn — a stack of stones used as a waymarker. CairnBooks helps you mark your financial trail.

CairnBooks is an open-source bookkeeping application designed for small businesses and independent operators. It provides a clean, modern interface for tracking income, expenses, invoices, and reports — without the complexity of enterprise accounting suites.
